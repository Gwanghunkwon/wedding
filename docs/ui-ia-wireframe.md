# GPT/제미나이 참고 UI-IA 와이어프레임

## 대화형 홈

- 상단 질문: "예식/인테리어 중 무엇을 준비 중인가요?"
- 단계형 입력: 카테고리 -> 지역 -> 인원/평형 -> 예산 -> 스타일
- 즉시 반응: 입력값 변경 시 우측 요약 카드 실시간 업데이트

## 비교 결과 화면

- 카드 1: 예상 총비용(평균/중앙/범위)
- 카드 2: 항목별 비중(예식: 식대/대관/스드메/옵션, 인테리어: 기본공사/옵션)
- 카드 3: 예산 대비 부족/여유 금액
- 카드 4: 근거 출처/표본수/갱신일

## 견적 검증 화면

- 견적 텍스트 붙여넣기 또는 파일 업로드(`.txt`, 이미지 OCR)
- 자동 항목 분류(식대, 대관료, 철거, 목공 등)
- 룰 기반 경고:
  - 평균 대비 25% 초과
  - 핵심 항목 누락
  - 계약시 주의 문구

## B2B 콘솔

- 리드 수신함(신규/응답완료)
- 리드 상세(요청 조건, 희망 예산, 문의 메시지)
- 응답 템플릿 빠른 전송
- 월간 성과(노출, 클릭, 문의, 응답률)
