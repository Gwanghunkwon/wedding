# 수집 소스 정책 매트릭스

| 소스 | 유형 | 허용 범위 | 금지 범위 | 비고 |
| --- | --- | --- | --- | --- |
| 참가격/통계청/공정위 API | 공공 | API 호출, 통계 필드 저장 | API 약관 위반 재배포 | 갱신주기 기록 필수 |
| 지자체 공개 공고/자료 | 공공 웹 | 공개 HTML 파싱(비개인정보) | 로그인 필요 페이지 수집 | robots.txt 확인 |
| 업체 공개 정가표 | 민간 공개 | 가격표/옵션표 구조화 추출 | 비공개 견적서 무단 수집 | 출처 URL 저장 |
| 블로그/공개 후기 | 민간 공개 | 가격 관련 문장/항목 추출 | 닉네임/연락처 저장 | 원문 최소 보관 |
| 사용자 제출 계약사례 | UGC | 금액/옵션/지역(시구) 저장 | 실명/전화번호/상세주소 저장 | 삭제요청 SLA 7일 |

## 준수 절차

1. 신규 소스 등록 전 정책 검토 체크리스트 완료
2. 월간 정책 점검(robots, 이용약관 변경 여부)
3. 비식별 필터 통과 후에만 서비스 DB 반영
4. 위반 의심 소스는 즉시 수집 중단 및 재검토
