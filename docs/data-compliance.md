# 데이터 수집 법적/정책 준수 가이드

## 원칙

- 공개 접근 가능한 데이터만 수집
- 개인정보 최소 수집/비식별 저장
- 출처별 이용약관/robots.txt 준수

## 소스별 정책

### 공공데이터

- 참가격/통계청/공정위/지자체 API 또는 공개 페이지 사용
- 가능하면 API 우선, HTML 파싱은 보조 수단
- 갱신 주기와 메타데이터(수집시각, 출처 URL) 기록

### 민간 공개 페이지

- 공개 블로그/사이트의 가격 정보만 수집
- 로그인/우회 접근/비공개 커뮤니티 데이터 수집 금지
- 본문 원문 저장 최소화, 필요한 필드만 구조화 저장

### 사용자 제출 데이터

- 계약 사례 등록 시 이름/연락처 저장 금지
- 주소는 시/구 단위로 일반화
- 삭제 요청 시 7일 이내 삭제 처리

## 금지 항목

- 비공개 카페/포럼 크롤링
- 기술적 우회(캡차 우회, 차단 회피) 수집
- 개인정보 직접 식별자 저장

## 운영 체크리스트

- [ ] 신규 수집 소스 등록 시 정책 검토 완료
- [ ] robots/이용약관 변경 모니터링
- [ ] 월 1회 샘플 데이터 법적 리스크 점검
- [ ] 데이터 삭제/정정 처리 SLA 준수
