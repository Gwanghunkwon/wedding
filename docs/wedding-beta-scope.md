# 웨딩 MVP 베타 범위 고정서

## 포함 범위 (Must Have)

1. 조건 기반 비교
   - 입력: 지역, 하객수, 예산
   - 출력: 평균 총액, 범위, 항목별 평균(식대/대관/스드메/옵션)
2. 사례 카드
   - 3~5개 사례 카드 노출
   - 출처/표본수/갱신일 표시
3. 견적 검증
   - 텍스트 + 이미지 OCR 입력 지원
   - 누락/주의 문구 탐지 + 금액 파싱 합계 기반 과다 경고
4. 업체 문의
   - 문의 메시지 전송
   - 리드 저장 및 B2B 수신함 연동
5. 이벤트 수집
   - `compare_started`, `compare_result_viewed`, `lead_submitted`

## 제외 범위 (Won't Have)

- 결제/계약 전자서명
- AI 생성 맞춤 리포트
- 실시간 동적 가격 예측
- 모바일 네이티브 앱

## 베타 합격 기준

- 핵심 페이지 4개(홈/웨딩비교/견적검증/B2B) 오류 없이 동작
- 문의 API 성공률 99% 이상
- 이벤트 누락률 5% 이하
