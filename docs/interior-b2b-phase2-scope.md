# 인테리어 확장 + B2B 수익화 2단계 범위

## 인테리어 확장 범위

1. 표준견적 엔진 고도화
   - 입력: 평형, 스타일, 공사범위, 자재 등급
   - 출력: 항목별 권장 단가, 총액 신뢰구간
2. 유사 계약사례 추천
   - 동일 평형/스타일의 최근 계약 사례 우선 노출
3. 시뮬레이션
   - 옵션 토글(철거/목공/조명/가구)로 실시간 비용 영향도 계산

## B2B 수익화 범위

1. CPL 과금
   - 유효 리드 기준 건별 과금
2. 월 구독 플랜
   - Basic: 리드 수신함/템플릿
   - Pro: 성과 리포트/우선 노출
3. 성과 대시보드
   - 노출, 클릭, 문의, 응답률, 계약등록 전환

## 릴리즈 조건

- 인테리어 비교→문의 전환률 KPI 대시보드 연결
- B2B 플랜 화면/과금 정책 문서 공개
